Basic Composition of an HTML Webpage

HTML code describes the structure of a web page. It consists of a series of elements that are defined by a start tag, the content, and a closing tag.

Empty elements do not need opening and closing tags, as there is no content to go in between them.

HTML elements can be classified into Block-level and Inline-level elements.

  • Block-level elements cause a line break to occur, and they take up space and stack down the page.
  • Inline-level elements, however, only take up space as is necessary.

The <!DOCTYPE html> declaration defines that this document is an HTML5 document. The <html> tag is the root of a HTML document and it should be added after !DOCTYPE declaration.

It consists of two main elements: the <head> and the <body> elements.

Common HTML elements

  1. Headings

    The heading tags represent all levels of headings in HTML. It has six levels, <h1> through <h6>, with <h1> denoting the most important with the largest font size and weight.

    • This is heading <h1>

    • This is heading <h2>

    • This is heading <h3>

    • This is heading <h4>

    • This is heading <h5>
    • This is heading <h6>

    Pro Tip: It is best to use <h1> tag per web page.

  2. Paragraphs

    The <p> tag creates a paragraph. Line breaks and whitespaces are usually ignored when paragraphs are rendered in a browser.

  3. Content Division Element

    The <div> tag represents a division or section in an HTML document, it serves as a container that can be used to group content so they can be easily styled.

    This styling can be done inline or by using CSS referencing their class or id attributes.

  4. Line Breaks

    The <br> tag is a good example of an empty element, it is used to enforce a line break within a <p> tag. It can be written as a self-closing tag <br /> or just the opening tag <br>.

Text Formatting

All HTML elements for formatting texts are an inline-level element. Some tags used in formatting text in HTML include the following.

  1. Strong text

    Code: <strong>This text is bold and important.</strong>

    This text is bold and important.

    Note that you can also use a <b> tag here.

  2. Emphasized Text

    Code: <em>This text is italicized for emphasis.</em>

    This text is italicized for emphasis.

    You can also use a <i> tag here.

  3. Small Text

    Code: <small>These words are smaller </small> than others.

    These words are smaller than others.

  4. Large Text

    Code:<big> These words are larger than </big> the others.

    These words are larger than the others.

  5. Highlighted Text

    Code:<mark> This text is highlighted.</mark>

    This text is highlighted.

  6. Strikethrough Text

    Code:<strike>This is an example of strikethrough text</strike>

    This is an example of strikethrough text

  7. Underlined Text

    Code:<u>This sentence is underlined.</u>

    This sentence is underlined.

  8. Superscript and Subscript Text

    Superscipt:An equation with superscript text: <sup>2</sup>.

    An equation with superscript text: X2.

    Subscript:We can also denote a chemical compound using the subscript tag:

    Code:CO<sub>2</sub>

    CO2

Links

The HTML <a> tag defines a hyperlink for navigation. It is an inline-level element. The href attribute holds the URL that will be navigated to.

This is <a href="url">a link</a>

Link Targets

The target attribute is used to specify the location document/URL would be displayed. Some of the possible options are:

  • _blank:opens the linked document in a new browser window or tab

    <a href="cheatsheet.html" target="_blank">link </a>

  • _self:opens the link in the same frame

    <a href="cheatsheet.html" target="_self">link </a>

  • _parent: opens the linked document in the parent frame

    <a href="cheatsheet.html" target="_parent">link </a>

  • _top: opens the document in the full body of the window

    <a href="cheatsheet.html" target="_top">link </a>

Special Links

<a href="mailto:email@example.com">Send email</a>

<a href="tel:0123492">Call 0123492</a>

Media Elements

You can add media elements to HTML, such as images, videos, and audio. Here’s how it’s done.

  1. Video

    Below, we add a video and its size to our web page. You can provide multiple sources and formats. The browser uses the first available one.

    <video width="300" height="240" controls>
<source src="url" type="video/mp4">
<source src="alternate-url" type="video/ogg">
</video>

  2. Audio

    <audio controls>
<source srce="url" type="audio/ogg">
<source srce="alternate-url" type="audio/mpeg">
</audio>

  3. Image

    <img src="path-to-image" alt="alternate-text" width="480px" height="360px">

Lists

There are several kinds of lists we can create with HTML.

  1. Ordered List

    <ol> defines an ordered list. It is numbered by default. The numbering format can be changed using the type attribute.

    <ol>
  <li>Water </li>
  <li>Tea</li>
  <li>Milk</li>
<ol>
    Result:
    1. Water
    2. Tea
    3. Milk

  2. Unordered List

    <ul> defines an unordered list. List items would appear bulleted.

    <ul>
  <li>Rice</li>
  <li>Vegetables</li>
  <li>Butter</li>
</ul>
    Result:
    • Rice
    • Vegetables
    • Butter

The HTML <form> element is used to collect and send information to an external source.

<form> can contain various input elements. When a user submits the form, information in these input elements is passed to the source which is named in the action attribute of the form.

Semantic HTML

Semantic HTML introduces meaning to the code we write. Before Semantic HTML the elements didn’t have any meaning as to what it does or what content goes in it.

An element such as <div> was used as a general-purpose element to create things from headers to footers to articles. With Semantic HTML we were introduced to elements that tell developers and browsers exactly what it does and what content should go in it.
